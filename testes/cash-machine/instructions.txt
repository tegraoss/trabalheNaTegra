Resolvi o cach-machine em python 3.5
Então para rodá-lo, pelo terminal vá até a pasta do arquivo "cachmachine.py"
Se estiver em uma distribuição ubuntu/debian o padrão é seguinte comando
$ python3 cashmachine.py
Em outras distribuições talvez seja necessário usar outro nome para chamar o interpretador de python 3+

Os testes devem ser rodados da mesma forma, logo:
$ python3 test.py

Assim que o programas estiver rodando ele aceitará uma entrada de um numero, responderá de acordo e finalizará sua execução.
